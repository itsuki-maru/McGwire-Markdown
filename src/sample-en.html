How to Use the Markdown Editor

Overview

Markdown is a means of creating documents like this. It is written according to the following rules. A characteristic of Markdown is that you can easily create well-formatted documents and meeting materials.

Document creation tools like Microsoft Word are highly functional, but they have a high learning cost and are complex, often resulting in different formatting depending on the creator.

McGwire Markdown is an application for creating documents in Markdown.

How to Create Headings

The size of the heading changes with the number of # characters. Leave a half-width space after the #. There are six levels of size. The # button (on the preview screen) can be clicked to insert it.

Heading1

Heading2

Heading3

Heading4

Heading5
Heading6

Example

# Heading1
## Heading2
### Heading3
#### Heading4
##### Heading5
###### Heading6

Emphasizing Text

Surround the **place to emphasize** with two *. The ** button (on the preview screen) can be clicked to insert it.

Example

**This place is very dangerous.**

Quotation

For quotations and double quotations (used in dialogues and minutes), use the > symbol. The > and >> buttons (on the preview screen) can be clicked to insert it.

Company ●●

Thank you for your continued support.

Regarding the issue you contacted us about the other day where you couldn't connect to the Internet.

There are days like that.

Example

> Company ●●
>
> Thank you for your continued support.
> 
> Regarding the issue you contacted us about the other day where you couldn't connect to the Internet.
>
> There are days like that.

For double quotations (like dialogues), use > and >> alternately.

Hello, Bob.

Hello, Alice.

Do you know where my car is?

Your car was driving now.

Example

> Hello, Bob.
>
>> Hello, Alice.
> 
> Do you know where my car is?
>
>> Your car was driving now.

How to Create a Bulleted List

For bulleted lists, use -. Leave a half-width space after the -. The - button (on the preview screen) can be clicked to insert it.

  • Author and Work
  • Keiichiro Yasuoka
    • Audubon's Prayer
    • Modern Times
  • A.D. Salinger
    • Catcher in the Rye

Example

- Author and Work
- Keiichiro Yasuoka
  - Audubon's Prayer
  - Modern Times
- A.D. Salinger
  - Catcher in the Rye

How to Create a Numbered List

For numbered lists, use 1.. Leave a half-width space after the 1.. The 1. button (on the preview screen) can be clicked to insert it.

  1. About Park Maintenance
    1. Safety Inspection of Playground Equipment
    2. Confirmation of Off-limits Areas
  2. About Construction Schedule
  3. Others

Example

1. About Park Maintenance
    1. Safety Inspection of Playground Equipment
    2. Confirmation of Off-limits Areas
2. About Construction Schedule
3. Others

Checklist

  • Checkbox 1
  • Checkbox 2

For checklists, use - [x] (checked) and - [ ] (unchecked). Leave a half-width space after the - [x]. The - [x] button (on the preview screen) can be clicked to insert it.

Example of description

- [ ] Checkbox 1
- [x] Checkbox 2

How to create a table

You create a table using the pipe | and hyphen -. The hyphen defines the headers of the columns, and the pipe separates the columns. You can insert this by clicking the | button (on the preview screen).

Title Content Notes
Catcher in the Rye Salinger None
Audubon's Prayer Kouki Isaka None
Modern Times Kouki Isaka None
Y Shogo Sato None

Example of description

| Title | Content | Notes |
| - | - | - |
| Catcher in the Rye | Salinger | None |
| Audubon's Prayer | Kouki Isaka | None |
| Modern Times | Kouki Isaka | None |
| Y | Shogo Sato | None |

Inserting links and images

Links are created in the form [link text](URL).

Example of link description

[Google](https://www.google.com)

Images are inserted in the form ![alt text](URL or file path of the image). You can insert an image by dropping it or by clicking the 📷 button (on the preview screen).

Example of image description

![logo](https://example.com/logo.png)

Page Break

When outputting to a PDF, you may want to insert a page break. In this case, you enter the following string.

<div style="page-break-before:always"></div>

You can insert this string by clicking the ↲ button (on the preview screen).
